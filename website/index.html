浏览器设置说明

由于安全策略，首次使用需要修改浏览器设置。如果不会改或者使用手机，请尝试点我进备用

Chrome / Edge 浏览器设置方法
  1. 在地址栏输入: chrome://flags/#unsafely-treat-insecure-origin-as-secure
  2. 在输入框中添加: http://172.16.1.4:8080
  3. 将下拉菜单改为 "Enabled"
  4. 点击右下角 "Relaunch" 重启浏览器
Firefox 浏览器设置方法
  1. 在地址栏输入: about:config
  2. 点击"接受风险并继续"
  3. 搜索: security.mixed_content.block_active_content
  4. 双击将值改为 false
  5. 重启浏览器
Safari 浏览器设置方法
  1. 点击"Safari" > "设置"
  2. 进入"隐私与安全性"
  3. 取消勾选"防止跨站跟踪"
  4. 重启浏览器
